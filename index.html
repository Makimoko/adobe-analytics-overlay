Installation du Bookmarklet

Faites glisser ce bouton dans votre barre de favoris, ou cliquez-droit => "Ajouter aux favoris"

Adobe Analytics Overlay
Important : Remplacez VOTRE-USERNAME par votre nom d'utilisateur GitHub dans le code du bookmarklet
dartyclic
Détecte automatiquement les liens avec ancre #dartyclic=nom_du_clic
Badge bleu
satellite.track('clic')
Capture les clics avec _satellite.track('clic') et extrait l'event_id
Badge rouge
satellite.track('interaction')
Identifie les interactions contextualisées avec données produit
Badge violet
Tag Commander
Accède aux variables tc_vars : page_pagename, env_template, product_category1, etc.
Overlay visuel
Encarts flottants avec les 4-5 valeurs clés : Type, Clic, event_id, Libellé, Page
Performance
Détection automatique au chargement et sur les mises à jour DOM (SPA/AJAX)

Exemples de détection

Testez le bookmarklet sur ces éléments (simulés pour la démo)

Lien dartyclic : Voir le produit
=> Détectera : Type: dartyclic, Clic: voir_produit
Détectera : Type: satellite.track(clic), event_id: open_menu
Détectera : Type: satellite.track(interaction), event_id: add_to_cart
Code JavaScript du Bookmarklet
javascript:(function(){const e=document.createElement('script');e.src='https://VOTRE-USERNAME.github.io/adobe-analytics-overlay/overlay.js?v='+Date.now();document.body.appendChild(e);})();
Déploiement GitHub
  1. Créez un repo adobe-analytics-overlay
  2. Uploadez le fichier overlay.js
  3. Activez GitHub Pages (Settings → Pages → main branch)
  4. Mettez à jour l'URL dans le bookmarklet
  5. Ajoutez le bookmarklet à vos favoris
Debug Console
// Vérifier l'état de l'overlay
window.AdobeAnalyticsOverlay.isActive()

window.AdobeAnalyticsOverlay.getTrackedCount()

window.AdobeAnalyticsOverlay.scan()

// Toggle manuel
window.AdobeAnalyticsOverlay.toggle()
